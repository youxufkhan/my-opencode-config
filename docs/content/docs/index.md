# Welcome to my-opencode-config

`my-opencode-config` is an interactive setup wizard that helps you quickly configure OpenCode with powerful free models, essential plugins, and optimal settings.

## Quick Start

Run the setup wizard directly without installation:

```bash
npx my-opencode-config
```

Or install globally:

```bash
npm install -g my-opencode-config
my-opencode-config
```

## What It Does

- **Intelligent Model Discovery** - Finds and recommends the best free models from OpenCode Zen
- **Essential Plugins** - Installs and configures popular plugins automatically
- **Superpowers** - Integrates advanced AI agent capabilities
- **Seamless Authentication** - Handles OAuth for OpenCode Zen and Google Gemini
- **Smart Configuration** - Generates opencode.json with optimal settings
